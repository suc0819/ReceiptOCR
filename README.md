# ReceiptOCR
### 영수증 OCR 기반 가계 관리 애플리케이션

2171435 최성욱

<br>

### 1. 프로젝트 수행 목적

#### 1.1 프로젝트 정의

* 영수증 OCR 기반 가계 관리 애플리케이션

#### 1.2 프로젝트 배경

* 기존 OCR 앱들은 단순 텍스트 추출 기능에 집중되어 있어 사용자가 필요한 핵심 정보를 직관적으로 확인하기 어렵다.
* 복잡한 UI와 과도한 기능으로 인해 사용성이 떨어지는 문제가 존재한다.
* 기존 OCR 서비스는 흐린 이미지, 영수증 특유의 작은 글씨로 인해 인식 정확도가 떨어지는 문제가 존재한다.

#### 1.3 프로젝트 목표

* 영수증 이미지에서 텍스트를 추출하고 상호명, 날짜, 총액 등의 핵심 정보를 자동 분석
* 이미지 전처리를 통한 OCR 인식 정확도 향상
* 분석 기록 저장 및 조회 기능 제공

### 2. 프로젝트 개요

#### 2.1 프로젝트 설명

* 사용자는 갤러리에서 영수증 이미지를 선택하여 텍스트를 추출할 수 있다.
* 흑백 처리, 대비 증가, 이미지 확대 등의 전처리 과정을 통해 OCR 인식 정확도를 향상시킨다.
* Vision Framework를 이용해 텍스트를 추출하고 상호명, 날짜, 총액을 자동 분류하여 직관적으로 제공한다.
* 분석된 결과는 UserDefaults에 저장되어 이후에도 조회할 수 있다.

#### 2.3 결과물

|------|--------|---------|

#### 2.4 기대효과

* OCR 기반 영수증 자동 분석으로 소비 내역을 간편하게 관리할 수 있다.
* 이미지 전처리를 통해 흐리거나 작은 글씨가 포함된 영수증도 안정적으로 인식할 수 있다.
* 유지보수와 확장이 용이한 MVC 구조로 설계되었다.

#### 2.5 관련 기술

| **구분** | **설명** |
|--------|--------|
| Vision Framework | Apple이 제공하는 이미지 분석 프레임워크로 OCR 텍스트 인식을 수행한다. |
| CoreImage | 이미지 전처리 및 보정 처리를 수행한다. |
| UserDefaults | 분석 결과를 로컬에 저장한다. |
